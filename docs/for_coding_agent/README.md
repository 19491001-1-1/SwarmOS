# For Coding Agents

This directory stores handoff material for future coding agents entering the Crewden repository.

Use this directory before starting implementation work:

1. Read `AGENTS.md` at the repository root.
2. Read the latest handoff document in this directory.
3. Read the relevant roadmap document under `docs/`.
4. Start from `main`, create a task branch, implement with tests, and merge only after verification passes.

## Documents

- [`session-handoff-2026-04-26.md`](./session-handoff-2026-04-26.md): current project state, recent work, common commands, operational notes, and known pitfalls from the recent coding sessions.

## What Belongs Here

- Session handoff summaries.
- Runbooks for recurring development or deployment operations.
- Lessons that are too operational for the product roadmap but useful for future agents.
- Cross-version context that helps an agent avoid rediscovering recent decisions.

## What Does Not Belong Here

- Secrets, tokens, API keys, or private credentials.
- Full raw chat logs when a concise summary is enough.
- Temporary scratch notes that will be stale after one task.
